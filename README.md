# FMML Projects and Labs — IIIT Hyderabad 🎓

This repository contains all lab notebooks and projects completed during the 
**Foundation of Modern Machine Learning (FMML)** course at **IIIT Hyderabad** 
as part of my AI/ML Research Internship (2024).

## What this covers
- **Linear Algebra** — mathematical foundations for ML
- **Module 1–6** — progressive ML concepts with hands-on lab work
- **Module Projects** — applied projects at the end of each module

## Tools and technologies used
- Python
- Jupyter Notebooks
- Google Colab
- NumPy
- Matplotlib

## About
This work was completed during my internship at **iHub Data, IIIT Hyderabad** — 
one of India's premier AI research institutes. The course covered foundational 
to intermediate machine learning concepts through theory and practical lab work.
